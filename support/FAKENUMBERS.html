Landline Numbers

Landlines use random digits attached to the prefixes 5550 or 7010. They can be used for each state or territory.

  • 🏙️NSW: (02) 5550 xxxx  or  (02) 7010 xxxx
  • 🌞QLD: (07) 5550 xxxx  or  (07) 7010 xxxx
  • 🏖️VIC: (03) 5550 xxxx  or  (03) 7010 xxxx

Replace each x with any digit. These numbers are for testing purposes only and should not be used for real communication.
